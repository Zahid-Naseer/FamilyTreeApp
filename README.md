# Family Tree App

A full-stack family tree web application built with Django.

## Features
- Private family groups with invite codes
- Visual family tree with SVG connector lines
- Zoom in/out with pinch-to-zoom mobile support
- Add children directly from the tree via modal
- Marriage management with age validation
- Role-based permissions and profile claiming
- PDF and Excel export
- Account settings with password change

## Tech Stack
- Python / Django
- SQLite
- HTML, CSS, JavaScript (vanilla)
- openpyxl, ReportLab
